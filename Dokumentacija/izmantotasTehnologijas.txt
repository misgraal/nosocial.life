IZMANTOTĀS TEHNOLOĢIJAS
======================

1) Backend
----------
- Python
- FastAPI
- asyncmy — asinhronam darbam ar MySQL
- passlib — paroļu hešošanai
- Jinja — HTML šabloni

2) Frontend
-----------
- HTML
- CSS
- JavaScript — File Explorer UI funkcionalitātei

3) Datu bāze
------------

--------------------------

- HTTPS piekļuve attālinātam savienojumam

-------------------
